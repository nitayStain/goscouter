# Standard Go Project Layout

goscouter/
├── cmd/
│   └── goscouter/           # Main application
│       ├── main.go
│       ├── version.go
│       └── version_check.go
│
├── internal/                # Private application packages
│   ├── server/
│   │   ├── router.go
│   │   ├── server.go
│   │   └── subdomain.go
│   └── subdomain/
│       ├── models.go
│       └── subdomain_finder.go
│
├── pkg/                     # Public library packages
│   └── placeholder/
│       └── doc.go
│
├── frontend/                # Next.js React frontend
├── scripts/                 # Installation scripts
├── go.mod
├── Makefile
└── README.md

Build commands:
- go build -o goscouter ./cmd/goscouter
- make backend
- make build (builds frontend + backend)
